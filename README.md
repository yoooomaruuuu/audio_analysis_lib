# audio_analysis_lib
## ビルド
* audio_analysis_lib.slnを開き、ビルドしてください
* \VS2019\x64\(Debug or Release)ディレクトリ内にlib_audio_analysis.dll及びwin.dllが出力されます。
    + win.dllはworldのdllです。
* 利用する際は其の2つをコピーしてください。
